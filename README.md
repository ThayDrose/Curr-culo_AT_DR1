##   Projeto de Currículo Profissional (Assessment HTML & CSS)

Este repositório contém o código-fonte de um **Currículo Profissional e Portfólio Pessoal** desenvolvido como um exercício prático e de avaliação (Assessment) das habilidades fundamentais em desenvolvimento web front-end.

---

###  Visão Geral

O objetivo principal deste projeto foi demonstrar proficiência na construção de uma página web responsiva e visualmente atraente utilizando apenas tecnologias base: **HTML5** e **CSS3**.

O foco do Assessment foi a **estrutura semântica** e a **estilização avançada**, garantindo um design limpo, moderno e otimizado para diferentes dispositivos.

###  Principais Pontos do Assessment

As seguintes áreas foram avaliadas e implementadas neste projeto:

* **Estrutura Semântica (HTML5):** Uso correto de tags como `<header>`, `<main>`, `<section>`, `<article>`, `<footer>`, e elementos de acessibilidade.
* **Layout e Responsividade (CSS3):**
    * Implementação de um design responsivo (`@media queries`) para garantir visualização perfeita em telas móveis, tablets e desktops.
    * Utilização de **Flexbox** e **CSS Grid** para layouts complexos (como a seção de Portfólio e a área Hero).
* **Estilização e Design:**
    * Uso de sombras (`box-shadow`), transições (`transition`) e efeitos visuais (como o **Degradê de Fundo** na seção de Habilidades).
    * Design de Cartões (Cards) e implementação de uma Timeline de Experiência.
* **Organização e Manutenibilidade do Código:**
    * Estruturação lógica do arquivo CSS (`styles.css`).
    * Aplicação de seletores eficientes (classes, IDs, e seletores de descendentes).
